Department OAuth Flow Service - Manages department-specific OAuth authentication flows.

Department OAuth Flow Service - Manages department-specific OAuth authentication flows. Provides OAuth integration with Google, Microsoft, and Apple for department-level authentication. Handles permission inheritance, context switching, and audit logging.
